## 🚀 مراحل نصب

```bash

git commit -m "Final version - Ready for Railway"
git push origin main
```

### 2. Deploy در Railway
1. بروید به [Railway.app](https://railway.app)
2. "New Project" کلیک کنید
3. "Deploy from GitHub repo" انتخاب کنید
4. Repository را انتخاب کنید
5. Deploy کنید
